\# 電流ログ解析ツール（Current Log Analyzer）



\## 概要

3系統電源の電流ログ（CSV）を解析し、  

定常状態における電流の安定性を数値およびグラフで評価するツールです。



評価業務で発生しがちな以下の課題に対応しています。



\- CSVヘッダ位置のずれ

\- 列名の表記ゆれ（例：Idc1(A) / Idc\_1(A)）

\- 立ち上げ過渡区間の除外

\- 欠損・未通電データの混入



---



\## 主な機能

\- CSVログの安全な読み込み

\- 実測データ特有の表記ゆれ対応

\- 評価開始時刻（開始◯秒後）の指定

\- 未通電・欠損区間の自動除外

\- 電流の統計量算出（min / max / mean / std）

\- 時系列グラフの自動生成

\- 評価条件を明記したレポート（Markdown）出力



---



\## 想定データ形式

以下のような列を含むCSVを想定しています。



\- Date

\- Time

\- Idc\_1(A), Idc\_2(A), Idc\_3(A)



※ 実データでは列名やヘッダ位置が異なる場合があるため、  

読み込み時に自動補正を行います。



---



\## 使い方



\### 1. ライブラリのインストール

```bash

pip install -r requirements.txt

---

\### 2. ダミーデータの生成（動作確認用）



実データを使用できない環境向けに、

電流ログのダミーデータ生成機能を用意しています。



python make\_dummy\_csv.py





input/sample.csv が生成されます。

---

\### 3. 解析の実行

python main.py --input input/sample.csv



---

オプション指定例：



python main.py --input input/sample.csv --skip\_sec 60 --min\_total 1.0





--skip\_sec : 評価開始までの除外時間（秒）



--min\_total : 合計電流がこの値以下の行を除外

出力内容



解析結果は output/ フォルダに出力されます。



summary.csv

各電源の統計量（min / max / mean / std）



report.md

評価条件および統計結果をまとめたレポート



plots/idc\_timeseries.png

電流の時系列グラフ（定常状態）



※ 画像・出力例はダミーデータを使用しています。

設計上の工夫点



ヘッダ位置が不定なCSVに対応するため、

先頭行を固定せず自動検出方式を採用



実測ログに多い列名の表記ゆれを吸収



評価条件をコードとレポートの両方に明示



実データを含めない構成とし、再利用性を重視

注意事項



本リポジトリには 実業務データは含まれていません。

機密情報・業務情報への配慮を前提とした構成としています。
